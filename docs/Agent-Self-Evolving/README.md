# Self-Evolving Agents Guide

An interactive, bilingual guide to the history, mechanisms, results, formulas, implementation patterns, and limitations of influential self-evolving agent research.

## Read the guide

**[Open the interactive website](https://autochip-lab.github.io/Knowledge-Hub/Agent-Self-Evolving/)**

The guide includes:

- A visual timeline of major capability shifts
- English and Chinese reading modes
- Interactive, draggable method-flow diagrams
- Reported benchmark results and paper links
- Formulas and highlighted implementation sketches
- Practical limitations and implementation checklists

## Papers and systems covered

Gödel Machines, POET, Enhanced POET, Reflexion, Voyager, STOP, GPTSwarm, ADAS, Gödel Agent, Darwin Gödel Machine, AlphaEvolve, and Hermes Agent Self-Evolving.

## Run locally

The site is static and has no build step. Open `index.html`, or serve the directory locally:

```powershell
python -m http.server 8000
```

Then visit `http://127.0.0.1:8000/`.

## Notes

Paper names and technical terms are retained where translation could reduce precision. Reported figures are presented as described in the corresponding papers; follow the linked sources for full experimental context.
